# Uppgift 9 – Avancerat (Frivillig)

# Teori
Kolla på video: [CSS Hover & Transitions](https://www.youtube.com/watch?v=zHUpQWgiVj0)
Läs mer: [CSS Gradients – MDN](https://developer.mozilla.org/en-US/docs/Web/CSS/gradient)

# Uppgifter
Använd dig av de avancerade CSS-teknikerna för att göra hemsidan mer interaktiv och visuell:

1. Gör så att minst ett element på hemsidan påverkas av `:hover`
   - T.ex. knappar, länkar, eller kort som ändrar färg eller storlek vid hover
2. Lägg till en gradient på något element på din hemsida
   - T.ex. bakgrund med `linear-gradient()` eller `radial-gradient()`

# Följande styling ska finnas med external CSS
- `:hover` pseudo-klass
- `linear-gradient()` eller `radial-gradient()`
- Gärna även `transition` för smooth-effekter

# Bonus idéer (om du vill göra mer)
- Lägg till `transition` för smooth-effekter
- Använd `box-shadow` för djup
- Lägg till CSS-animationer med `@keyframes`
